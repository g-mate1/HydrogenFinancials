# HydrogenFinancials

Financial profiles of public companies exposed to the hydrogen economy.
One markdown file per company in `companies/`.

## Scope

Phase 1 covers ten hydrogen pure-plays and dedicated fuel-cell /
electrolyser specialists. Future phases may add industrial-gas majors
(Linde, Air Liquide, Air Products), H2 offtakers (voestalpine, Thyssen-
krupp Steel), and emerging mobility / integration players.

## Methodology

- **Individual research per company.** Each company profile is
  researched via direct web search against primary sources: company
  IR pages, annual reports, earnings releases, and stock exchange
  filings (SEC 10-K / 20-F, LSE, Euronext, OSE, Deutsche Börse,
  Nasdaq Stockholm). No batch scripts, no synthetic data.
- **Primary sources only.** Secondary sources (news, analyst notes)
  are cited only where they provide information not in filings.
- **"Not disclosed" over guesses.** If a figure is not publicly
  reported, the file records "not disclosed" rather than an estimate.
- **Latest full fiscal year plus latest reported quarter.** Figures
  in reporting currency.
- **Links in every citation.** Every material number is followed by
  a source URL.
